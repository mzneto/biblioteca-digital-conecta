# Biblioteca Digital Conecta

Projeto de banco de dados relacional para a Biblioteca Digital Conecta.

## Estrutura do Projeto

- **sql/create_tables.sql**: criação das tabelas.
- **sql/insert_data.sql**: scripts para popular as tabelas.
- **sql/select_queries.sql**: consultas SELECT.
- **sql/update_data.sql**: comandos UPDATE.
- **sql/delete_data.sql**: comandos DELETE.

## Instruções

1. Criar o banco de dados no MySQL/PostgreSQL.
2. Executar o script `create_tables.sql`.
3. Inserir dados com `insert_data.sql`.
4. Testar consultas com `select_queries.sql`.
5. Atualizar e remover dados com `update_data.sql` e `delete_data.sql`.

## Observações

- As tabelas possuem chaves primárias e estrangeiras.
- Os scripts respeitam as formas normais aplicadas no projeto.
